¿Qué es la verificación de propiedad?

La verificación de propiedad de un sitio web es un proceso que demuestra que eres el propietario legítimo de un sitio web. Al verificar la propiedad de un sitio web le estas indicando a Seo WebMas que eres su propietario, obteniendo asi acceso a datos privados e información sencible de Seo Webmas sobre el sitio web verificado.Los propietarios de sitios web también pueden realizar acciones que pueden afectar a la presencia y al comportamiento de un sitio. Por este motivo, queremos asegurarnos de que solo los propietarios reales de sitios web tengan permisos de propietario en Seo WebMas.

¿Cuánto tiempo es válida la verificación?

La verificación es válida mientras la suscripción del sitio web esté vigente y mientras Seo WebMas pueda confirmar la presencia y validez del token de verificación ya sea mediante un método de verificación u otro

¿Qué métodos de verificación tiene Seo WebMas?

Seo WebMas cuenta con dos métodos de verificación: verificación mediante etiqueta meta y verificación mediante subida de archivo HTML. La verificación mediante etiqueta meta es un método sencillo que requiere la capacidad de editar el código fuente HTML de la página principal de tu sitio. Por otra parte, para la verificación de propiedad mediante subida de archivo HTML debes subir un archivo y publicarlo en tu sitio a través de una URL concreta.

¿Cómo verfico mi sitio mediante etiqueta meta?

Verifica que eres el propietario de un sitio añadiendo una etiqueta <meta> al código HTML de una página concreta. Seo WebMas comprobará que la etiqueta meta esté presente en esa página. Esta etiqueta está vinculada a un usuario concreto.

Requisitos:

  • En la página principal no se puede requerir la autenticación. Los usuarios deben poder acceder a la página principal del sitio sin tener que iniciar sesión.
  • Debes poder editar el código fuente HTML de la página principal del sitio directamente. En concreto, debes poder insertar una etiqueta en la etiqueta <head> de tu página principal.
